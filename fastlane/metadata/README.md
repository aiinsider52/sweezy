Screenshots
-----------
Create PNGs for each locale under:
- fastlane/metadata/<locale>/screenshots/

Suggested iPhone sizes (portrait):
- 1290×2796 (6.7")
- 1179×2556 (6.1")
- 1242×2688 (6.5")
- 828×1792 (6.1" LCD)

Then upload with:
  bundle exec fastlane ios metadata
